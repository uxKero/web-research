# Contributing

This is a template, so the best contribution is forking it and making it yours.

If you want to improve the template itself:

1. Fork and branch.
2. `pnpm install` (Node 24+), make your change.
3. `pnpm typecheck && pnpm build` must pass.
4. Open a PR with a clear description.

Keep it focused and minimal, the point of the template is to be easy to read and adapt.
